# contrib-sumo-focal
Micro add-on for 5way navigation for SumoJS, use dist/focal-min.js for stand-alone use.

Usage, define focal containers
```html
<div class="focalContainer">
</div>
```

define focals!
```html
<div class="focal">
</div>
```

put them together!
```html
<div class="focalContainer">
<div class="focal">
</div>
<div class="focal">
</div>
<div class="focal">
</div>
<div class="focal">
</div>
</div>
```

You just made some 5way spottable controls! Way to go you!
